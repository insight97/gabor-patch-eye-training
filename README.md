# Gabor Patch Eye Training

Gabor patch（蓋博符號）是一種明暗相間的條紋圖案，這類圖案能有效刺激視覺皮質，幫助大腦更有效率地處理影像。研究指出，持續練習有機會提升視力表現，並改善對比敏感度、細節辨識與視覺反應速度，讓看東西更清楚、更快。

## 專案結構

- `index.html`：頁面結構與主要區塊
- `styles.css`：介面樣式
- `app.js`：刺激生成、互動流程、計分、歷史紀錄
- `docs/`：補充文件

## 注意

- 本專案為一般視覺練習用途，非醫療診斷或治療工具。
- 實際訓練體驗會受到螢幕亮度、尺寸、解析度與觀看距離影響。
